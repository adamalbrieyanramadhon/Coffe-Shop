Kopiku - Premium Artisan Coffee Shop Template

Thank you for purchasing the Kopiku template. This template is designed for coffee shop owners and local businesses who want a modern, responsive, and aesthetic website with a built-in Direct Order to WhatsApp feature—no complex backend or database setup required.

Key Features
- Modern & Aesthetic Design: A clean, artisan-inspired look perfect for trendy coffee shops.
- 100% Mobile Responsive: Fully optimized for all devices, from smartphones to desktops.
- Direct WhatsApp Ordering: Interactive pop-up form that automatically sends orders to your WhatsApp number.
- Zero Maintenance: Built with pure HTML, CSS, and Vanilla JavaScript.

Installation & Customization Guide

1. Set Your WhatsApp Number
Open the index.html file using any text editor. Search for the following line:
const noWhatsApp = "6281234567890"; 
Replace 6281234567890 with your actual WhatsApp number. Save the file (Ctrl + S).

2. Customize Content
- Shop Name: Search for Kopiku. in the header section and update it to your brand name.
- Menu Items: Within the index.html file, locate the menu-card sections. You can easily modify the item name, description, and the price to match your own menu.

3. How to Host Your Website
Since this is a static website, you can host it for free using:
- GitHub Pages: Push your files to a GitHub repository and enable "Pages" in the settings.
- Netlify/Vercel: Simply drag and drop your project folder onto their dashboard, and your site will be live instantly.
